## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

Uses `ai_impact_student_performance_dataset.csv` in this folder.

### Notebook (recommended)

```bash
source .venv/bin/activate
jupyter notebook student-ai-usage-analysis.ipynb
```

### Python script

```bash
python ai_usage_analysis.py
```

Outputs are written to `output/`:

- **`analysis_report.html`** — visual summary of the full usage analysis (opens in your browser automatically)
- PNG charts (`01`–`09`) and `processed_students_with_alerts.csv`

## Dataset

`ai_impact_student_performance_dataset.csv` — 8,000 student records with AI usage, study habits, and performance metrics.

Derived fields (not in the raw CSV) used for analysis:

- **subscription_type** — tier from `ai_dependency_score` (Free / Basic / Premium)
- **session_duration_minutes** — `ai_usage_time_minutes` / `ai_prompts_per_week`
- **missed_deadlines** — proxy from consistency and attendance

## Scenario 3 — Student Performance Prediction System

An end-to-end ML pipeline that predicts student academic performance and dropout
risk, then recommends evidence-based interventions for at-risk students.

### Systems

1. **Exploratory Data Analysis** — distributions, correlations, and visual profiling
2. **Performance Tier Predictor** — Random Forest classifies GPA tier (A/B/C/D/F), ~77% accuracy
3. **Dropout Risk Engine** — Gradient Boosting predicts Low / Moderate / High risk, ~79% accuracy
4. **Intervention Recommender** — rule-based matching of at-risk students to interventions, plus effectiveness analysis of past interventions
5. **Interactive Dashboard** — `ipywidgets` sliders to predict a hypothetical student's outcome live

### Run

```bash
source .venv/bin/activate
jupyter notebook Student_Performance_Prediction_System.ipynb
```

System 4 writes `dataset/intervention_plan.csv` (high-risk students and their
recommended interventions).

### Datasets (`dataset/`)

- **`student_performance_data.csv`** — 1,500 student records with behavioral and socioeconomic features, `current_gpa`, `performance_tier`, and `dropout_risk`
- **`intervention_log.csv`** — 743 historical intervention records with outcomes (Improved / No Change / Declined)
- **`intervention_plan.csv`** — generated by the notebook; recommended interventions for predicted high-risk students
